# Lacerda Barbershop

Monorepo da **Lacerda Barbershop**, desenvolvido com **Turborepo**, reunindo as
aplicações e pacotes do ecossistema do projeto em um único repositório.

## 📦 Estrutura do projeto

Este monorepo foi criado para centralizar os sistemas da barbearia, facilitando
manutenção, escalabilidade e reaproveitamento de código.

Atualmente, o projeto engloba:

- **Landing Page** Página institucional da barbearia, com apresentação dos
  serviços, marca e presença online.

## 🛠️ Tecnologias

Este projeto utiliza uma arquitetura moderna para desenvolvimento web, com foco
em organização e produtividade.

Principais ferramentas:

- **Turborepo**
- **TypeScript**
- **Next.js**
- **Tailwind CSS**
- **Shadcn/UI**

## 🧱 Arquitetura

A proposta do monorepo é separar aplicações e pacotes compartilhados de forma
clara.

```bash
└── lacerda-barbershop/
    ├── apps/
    │   └── landing
    └── packages/
        ├── eslint
        ├── tsconfig
        └── ui
```
